# Portfolio — Heni Abid
## Structure
- index.html   → Page principale du portfolio
- style.css    → Tous les styles (template Luminary + personnalisations)
- script.js    → Interactions, animations, compteurs
- images/      → Images du template original

## Hébergement
### Netlify Drop (30 secondes)
1. Allez sur https://app.netlify.com/drop
2. Glissez-déposez ce dossier entier
3. Votre site est en ligne !

### GitHub Pages
1. Créez un repo : heniabid.github.io
2. Uploadez tous les fichiers
3. Settings → Pages → Deploy from main
